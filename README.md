# Traffic Light Controller using Verilog HDL

## Project Overview
This project implements a Traffic Light Controller using Verilog HDL. It demonstrates the design of a Finite State Machine (FSM) for controlling traffic signals.

## Features
- Verilog HDL implementation
- FSM-based design
- Timer-based traffic light control
- Simulation using Xilinx ISE

## Tools Used
- Xilinx ISE Design Suite
- Verilog HDL

## Files
- traffic_light_controller_timer.xpr
- traffic_light_controller_sensor.xpr
